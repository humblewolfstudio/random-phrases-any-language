# random-phrases-any-language

## Features
- Returns a random text with x phrases and in any language supported by the ISO language codes.

## Install
```shell
npm install random-phrases-any-language
```

## Usage
```js
import { generateAndTranslateText } from 'random-phrases-any-language';

var text = await generateAndTranslateText(10, 'es');
```
